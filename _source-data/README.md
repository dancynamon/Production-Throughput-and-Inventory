# Drop your source files here

Put any existing spreadsheets, parts lists, BOMs, supplier sheets, etc. in this
folder and tell me they're here. I'll read them and map them into the app's
Google Sheet (`Products`, `RawMaterials`, `BOM`, `Employees`, `Planning`).

**Anything in this folder is git-ignored** (see `.gitignore`), so it will NOT be
committed to the repo or exposed on GitHub Pages. It stays private to our
working session.

Handy formats: `.xlsx`, `.csv`, `.ods`, `.pdf`, images of a printed list — all
readable. If it's already a **Google Sheet**, you don't need to drop anything
here at all; just tell me its name and I can read it straight from your Google
Drive (that account is connected to this session).
